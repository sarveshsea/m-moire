v0.12.4
Publish Figma and tweakcn design systems as installable registries. Ship tokens and real components to npm, then install them into any shadcn app with one command.
Start with Figma or tweakcn, publish with memi publish or memi theme publish, then install with memi add. Primary package: @sarveshsea/memoire.
About
Memoire is a registry-first CLI and MCP server. Publish design systems from Figma, tweakcn, or extracted site docs; install real components into shadcn apps; and keep the same workflow available to AI tools only after the registry loop is in place.
Specs
Structured component, page, and data visualization specs validated with Zod schemas. Every component starts as a spec before code generation.
Codegen
Generates production React components using shadcn/ui and Tailwind exclusively. Outputs to atomic folders: atoms, molecules, organisms, templates.
Research
Synthesizes user research from Excel files, web sources, and Figma stickies into actionable insights, themes, and personas.
Tokens
Extracts design tokens from Figma -- colors, typography, spacing, and effects -- and maps them to your codebase's token system.
Preview
Live component gallery served on localhost. Browse specs, research data, design system tokens, and generated components in one place.
Agents
Autonomous design agents powered by Claude. Classify intent, plan execution, generate artifacts, and self-heal after every canvas operation.
